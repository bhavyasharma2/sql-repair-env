Four tasks

Each task gives the agent a broken query, a schema, sample data, and a plain-English objective. The grader is fully deterministic.

EASY
Syntax repair
Fix a missing comma and a misspelled keyword (FORM → FROM) in a simple employee SELECT query.

MEDIUM
Wrong JOIN semantics

HARD
Window function bug
Fix a ROW_NUMBER() window function partitioned by the wrong column — returns wrong top-product per region. Query runs cleanly.
silent bug
EXPERT
Broken recursive CTE
Fix two bugs in a recursive CTE traversing an org chart — wrong anchor condition and wrong recursive join. Frontier models rarely solve this in one shot.
two bugs
Shaped reward signal

Partial credit at every step

The agent receives a gradient signal even before fully solving the task. Four components sum to exactly 1.0.

+0.20
Syntax valid

  "row_count_correct": 0.30,  # right # of rows

}
